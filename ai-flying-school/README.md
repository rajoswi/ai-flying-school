# AI Student Metrics & Safety Early Warning System - Hackathon Repo

This repository is a prototype for detecting 'at-risk' cadets in a flying school.

## Contents
- `backend/` - FastAPI backend, training script, model, sample CSV
- `frontend/` - Minimal React single-file App.jsx (Vite)
- `deployment/` - Docker Compose for running both services
- `cadet_weekly.csv` - synthetic sample data (50 rows)
- `model.joblib` - generated by running train.py (created below)

## Quick start (local)
1. Backend:
   - `cd backend`
   - `pip install -r requirements.txt`
   - `python train.py`  # creates model.joblib
   - `uvicorn app:app --reload --host 0.0.0.0 --port 8000`

2. Frontend (dev with Vite):
   - `cd frontend`
   - `npm install`
   - `npm run dev -- --host 0.0.0.0`
   - open the Vite URL (default http://localhost:5173)

3. Or run with Docker Compose (requires Docker):
   - `docker-compose up --build` from repository root

## Notes
- The label generation in `train.py` is rule-based for demo purposes.
- For production, replace the model, add auth, and use persistent DB.
